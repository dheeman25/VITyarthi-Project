**#CalorieTracker App**





**\*Overview\***

*A simple Python based Application that will help people set their daily calorie limits and meet their dietary goals.*





**\*Features\***

* *set your daily calorie limit*
* *measure your calorie intake*
* *assign your meal according to the data*
* *view daily progress and statistics*
* *simple user interface* 
*  *Expert Based recommendations*







**\*Technology stack\***

* *Python coding language*
* *simple file storage(JSON/CSV FILE)*
* *Devlopement Enviorment: VS CODE*
* *basic design interface(Upgradable)*
